# ITQN Mobile App — Hỗ trợ

Phiên bản ứng dụng: **1.x.x**

## Liên hệ hỗ trợ

Nếu bạn gặp vấn đề khi sử dụng ứng dụng, vui lòng gửi email tới:

**support@itqn.vn**

Khi gửi yêu cầu hỗ trợ, bạn vui lòng kèm theo thông tin:

- phiên bản app
- nền tảng (iOS / iPadOS)
- mô tả lỗi
- các bước tái tạo lỗi
- screenshot nếu có

## Câu hỏi thường gặp

**Làm sao đăng nhập?**  
→ Mở app → màn hình Login → nhập email + mật khẩu → Đăng nhập

**Quên mật khẩu?**  
→ Dùng chức năng *Forgot Password* tại màn hình đăng nhập
